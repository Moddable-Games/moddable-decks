# News & Post Templates

The news index + a single post template, lifted from moddable.games/news/.

## Files
- `index.html` — News index. Featured post on top, then 2-col grid of posts. Sidebar with Hot Topics (categories) and News Archives (months).
- `post.html` — Single news post template — "Are you ready to nuke Catan?" — long-form with sticky author rail.
